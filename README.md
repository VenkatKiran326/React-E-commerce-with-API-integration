# 🎄 Christmas Eve E-Commerce Website

This project is a **Christmas Eve themed E-Commerce Web Application** built using modern React technologies.  
The UI design inspiration was taken from a **ThemeForest template**.

I built this project to practice **real-world React development concepts** such as routing, API integration, and dynamic UI rendering.

---

## 🚀 Technologies Used

- React.js
- JavaScript
- React Router DOM
- JSON Server (temporary backend)

---

## ✨ Features

- Product listing page
- Product detail page
- Dynamic routing
- Nested routing
- API data fetching
- Error handling
- Responsive UI

---

## 📚 What I Learned

Through this project, I improved my understanding of:

- API Integration
- Dynamic Routing
- Nested Routing
- Building Product Detail Pages like real e-commerce apps
- Passing dynamic data from JSON
- Proper usage of `useEffect`
- Implementing **Error Boundaries** for better error handling

---

## 💡 Key Takeaway

One of the things I’m most proud of is that I built this project **without using any AI tools**.  
I wanted to challenge myself to think independently, debug errors, and truly understand how everything works under the hood.

This project helped strengthen my knowledge in:

- Component-based architecture
- React fundamentals
- State management basics
- Route structuring in larger applications
- Handling real-world UI scenarios
